============================================
Smart Budget Buddy - Personal Finance Tracker
============================================

How to run:
  python main.py

This is a GUI application using tkinter.
It will NOT run on Ed's environment.
Please run it locally on your computer.

External library required:
  - pandas (install with: pip install pandas)

Built-in libraries used:
  - tkinter
  - os
  - datetime

Project files:
  - main.py          (Main script - run this file)
  - login_page.py    (Login and register page)
  - budget_page.py   (Budget management page)
  - data_manager.py  (Data storage and processing)
  - user_data.csv    (User account data)
  - data/            (Folder for user budget records)
